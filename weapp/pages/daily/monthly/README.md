# 每月还款笔记
> 初衷：想要一个记录每个月还款（大笔且年限较长贷款，如房贷、车贷、大额消费贷）金额，每个月初自动生成当月还款清单，还了之后打钩

## 数据结构
> 保存到文件

包含`还款计划`、`每月还款历史`两种数据，示例：

```javascript
{
    "loan": [
        { 
            name:"",                //贷款内容
            begin:"",                 //开始日期
            end:"",                   //结束日期
            amount: 10000,       //贷款总额
            day:15,                   //账单日
            value:1000,             //每期还款金额
        }
    ],
    "history": {
        "202101" : { total: 1000, items: {"房贷":1000}}
    }
}
```
